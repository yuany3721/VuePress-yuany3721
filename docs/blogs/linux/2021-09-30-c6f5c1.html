    Ubuntu startup设置开机自启命令

    yuany3721 2021/9/30 startupubuntu

    Version: Ubuntu Server 20.04.3 LTS

    1. 建立自己的脚本，如wlt.sh
    2. 修改脚本执行权限chmod 775 wlt.sh
    3. 将脚本放入/etc/profile.d路径下即可 /etc/init.d update-rc.d wlt.sh defaults 91
